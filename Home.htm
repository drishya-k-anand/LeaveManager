Time Table Management

The Time Table Management provides Users with an Graphical User Interface to Manage their Time Tables. Users are also provided with screens to create the Time Table Entries with Particulars such as the Tutor/User Name, Subject, Day of Week, Start/End Time, Semester Start/End Month and so on.

Leave Approval Management

The Approvers can use an Interactive Dashboard that allows them to Accept and Decline Leave Requests. There are screens that allows Users to configure the Approval Persons/Users for each respective Tutor/User.

Substitue Allocation

Allows Users to allocate Substitue Tutors on the day of the Leave with all the necessary particulars such as the Substitute Tutor/User, Subject, Date and the Start/End Time.
